大先生討伐戦 (Operation: Fact Check) - アプリケーション説明書

1. ゲーム概要

オフィスに蔓延る「曖昧さ」「責任転嫁」「やったふり」を具現化したボスキャラクター**「大先生」**と戦う、ブラウザ動作の3Dシューティングゲームです。
プレイヤーは勇敢な部下となり、「事実確認」や「納期」といった概念を弾丸として撃ち込み、プロジェクトの平和を守る（＝大先生を退場させる）ことを目指します。

2. ストーリー

とあるプロジェクトの会議室（電脳空間）。
そこには、自信満々に振る舞いつつも実務を行わず、曖昧な言葉で周囲を煙に巻く「大先生」が君臨していた。
彼の口癖は「ちょうどやろうと思ってました」「ボールは向こうにあります」。
プレイヤーは彼を問い詰め、その「進捗率（HP）」を物理的に削りきり、彼を定時退社（昇天）させなければならない。

3. キャラクター設定：大先生

性格:

自信過剰で話が長い。

責任は取りたくないが、良い顔はしたい。

悪気はなく、本人は「ちゃんとやっている」つもり。

攻撃パターン（口撃）:

「認識のズレですね」

「アグリーです」

「一旦持ち帰ります」

「承知です」（NEW）

「娘の送迎がありますのでこの辺で業務終了します」（NEW）

「今日は分断で業務させていただきます」（NEW）

特殊行動:

逃走モード: HPが30%を切ると、「次の会議」などを理由に赤くなって高速で逃げ回り始める。

5. ユニークな機能

オリジナル大先生生成システム:

ユーザーが任意の顔写真をアップロードすると、自動的に丸く切り抜かれ、ボスキャラクターのテクスチャとして適用されます。

Canvas自動描画: 画像をアップロードしない場合、プログラムコード（Canvas API）によって描かれた「眼鏡をかけた自信ありげな大先生」のイラストが自動生成されます。

勝利演出:

HPを削りきると、大先生は光に包まれ、回転しながら天へと昇っていきます（定時退社）。同時に盛大な紙吹雪が舞い、ミッションコンプリートとなります。

6. 技術スタック

言語: HTML5, CSS3, JavaScript

ライブラリ: Three.js (WebGL 3D描画)

構成: 全てのコードが単一のHTMLファイルに含まれており、ブラウザさえあればインストール不要で動作します。
